# The TopoToolbox Gallery

This repository hosts a collection of example workflows as interactive notebooks.

The gallery itself can be viewed at https://topotoolbox.github.io/gallery.

The templates for
[MATLAB](https://topotoolbox.github.io/gallery/notebooks/matlab/contributing/contributing.html)
and
[Python](https://topotoolbox.github.io/gallery/notebooks/python/contributing/contributing.html)
contain detailed instructions for creating and submitting an example
to the gallery. In short, clone the gallery repository, create your
Jupyter notebook or MATLAB Live Script in a new directory under the
`notebooks/python` or `notebooks/matlab` and make a pull request to
add the example to the repository.
